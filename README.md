🧮 Calculadora Simples

Este projeto foi desenvolvido como um mini projeto de um curso, com o objetivo de praticar conceitos fundamentais de desenvolvimento web.

A aplicação consiste em uma calculadora simples, capaz de realizar operações básicas, servindo como exercício prático de lógica e interface.

🚀 Objetivo
Praticar HTML, CSS e JavaScript
Desenvolver lógica de programação
Trabalhar com manipulação de eventos e interações
🛠️ Tecnologias utilizadas
*HTML5
*CSS3
*JavaScript

⚠️ Observação

Este é um projeto de aprendizado, portanto podem existir pequenos erros ou melhorias a serem feitas.
